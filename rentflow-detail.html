Rentflow - ERP Prototype

ระบบบริหารจัดการหอพักและคอนโดมิเนียมแบบครบวงจร พัฒนาด้วย Google Apps Script

รูปภาพโครงการ (Gallery)

1. มุมมองเจ้าของหอ (Owner View)

แสดงหน้า Dashboard หลักสำหรับบริหารจัดการ, การจัดการห้องพัก, และระบบสร้างใบแจ้งหนี้อัตโนมัติ

2. มุมมองผู้เช่า (Tenant View)

แสดงหน้าจอการเข้าสู่ระบบ, การชำระใบแจ้งหนี้, ประวัติการชำระเงินในฝั่งผู้เช่า (ยังไม่ได้ทำ), และแจ้งปัญหา (ยังไม่ได้ทำ)

ที่มาและความสำคัญ (Problem)

ปัญหาเดิมของการจัดการหอพักคือการบันทึกข้อมูลด้วยมือ (Manual) ทำให้เกิดความผิดพลาดในการคำนวณค่าน้ำ-ค่าไฟ และความล่าช้าในการจัดส่งใบแจ้งหนี้ให้ผู้เช่า ผมจึงออกแบบระบบนี้เพื่อลดขั้นตอนที่ซ้ำซ้อนและเพิ่มความแม่นยำของข้อมูล

การแก้ปัญหา (Solution)

  • Automation: ระบบสร้างใบแจ้งหนี้อัตโนมัติจากฐานข้อมูล Google Sheets
  • Centralized Data: รวมศูนย์ข้อมูลสัญญาเช่าและสถานะการจ่ายเงินไว้ที่เดียว

Tech Stack

JavaScript (GAS) HTML5 CSS3 Google Sheets API
